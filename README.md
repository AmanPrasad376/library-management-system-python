# Library Management System (Python)

This is a menu-driven Library Management System developed using Python. The project allows users to manage books and members in a simple command-line interface.

## Features
- Add new books to the library
- Add and manage library members
- Modify book information
- Modify member information
- Issue books to members
- Return books to the library
- Search books or members
- View stored records using file handling

## Technologies Used
- Python
- File Handling
- Lists and Functions
- Conditional Statements and Loops

## How to Run the Project
1. Download the Python file.
2. Open the file in any Python IDE or terminal.
3. Run the program.
4. Use the menu options to manage library operations.
